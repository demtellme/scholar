## Scholar
Terminal based education assistant to log tests and help set targets 
Comptible wit todo to-do list (https://github.com/demtellme/todo.git)
This program is mainly made for personal use but made avaliable for practice
The commands to use the program is longer than others as its not meant to be used crazy often and it makes you more concious of what youre actually doing - trying to get better scores  
Wont be updated or developed much, basically abandoned 

## Installation

```bash
# Install with pipx (recommended)
pipx install git+https://github.com/demtellme/scholar.git

# Or with pip
pip install --user git+https://github.com/demtellme/scholar.git
```

## Uninstall Command
```bash
pipx uninstall scholar
```
## How to use
```bash
scholar <command>
```
 ## Commands:
    log - logs an exam, what went well and what can be done better
    fback - Displays what can be improved without logging it anywhere
    tgt - adds a target to a text file that you can see what should be improved, if todo to-do list is active it uses the list
